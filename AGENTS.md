### 📝 TYPESCRIPT

- **NO `any`**: USE `unknown`, GENERICS, PROPER TYPES
- **NO `enum`**: USE `const enum` OR `as const`

- **NEVER DESTRUCTURE PROPS**: KILLS REACTIVITY. USE `splitProps` OR `props.property`
- **PROPS ARE GETTERS**: ALREADY REACTIVE
- **STATE**: USE `batch(() => ...)` FOR MULTI-UPDATES

- **ENGINE**: TAILWIND CSS V4
